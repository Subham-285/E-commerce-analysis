# E-commerce Sales Power BI Dashboard

## Project Overview
This project showcases an interactive E-Commerce analytics dashboard built using Power BI. The dashboard provides a comprehensive view of sales performance, customer behavior, and product trends to support data-driven business decisions.

It is designed with a clean UI and focuses on key KPIs, trends, and comparisons commonly used by business stakeholders and decision-makers.

## Business Insights
- Laptops generate the highest revenue despite lower order volume
- Smartphones and Tablets lead in total orders
- Strong sales contribution from North America and Europe
- Balanced payment method usage across customers
- AOV trends help identify premium vs high-volume products

## Tools & Technologies
- Power BI (Data modeling, visualization, and reporting)
- DAX (Data Analysis Expressions) (Creating calculated measures and KPIs)
- Power Query: Data Cleaning & Transformation
- Excel / CSV: (Managing and importing project and budget data)
